# PropertyApp Brand System — prompt.md

## 1. Brand Overview

**Brand Name:** PropertyApp  
**Tagline:** Your Property In Right Hands  

### Core Concept
PropertyApp is a modern digital real estate platform focused on trust, simplicity, and structured property management. It represents reliability, clarity, and professional control over property transactions.

### Brand Personality
- Trustworthy
- Professional
- Minimal
- Structured
- Modern
- Calm and reliable

### Brand Tone
- Clear and direct
- Confident but not aggressive
- Professional and composed
- No exaggeration or hype language

---

## 2. Logo System

### Logo Types
- Primary Logo (Wordmark)
- Monomark (Icon)
- Colourmark (Full color)
- Whitemark (White version for dark backgrounds)

### Logo Rules
- Maintain clear spacing around logo (minimum = 2X unit)
- Do not stretch, rotate, or distort
- Do not change colors outside brand palette
- Use Whitemark on dark backgrounds only
- Use Colourmark on light backgrounds

### Spacing System
- Define spacing unit: X
- Minimum safe area: 2X on all sides

---

## 3. Color System

### Primary Colors

#### Deep Blue (Primary)
- HEX: #02376D
- RGB: (2, 55, 109)
- CMYK: (100%, 86%, 31%, 18%)

**Usage:**
- Main backgrounds
- Headers
- Primary buttons
- Key UI elements

---

#### White (Base)
- HEX: #FFFFFF
- RGB: (255, 255, 255)
- CMYK: (0%, 0%, 0%, 0%)

**Usage:**
- Backgrounds
- Text contrast
- Clean spacing

---

#### Accent Red
- HEX: #F84235
- RGB: (248, 66, 53)
- CMYK: (0%, 89%, 84%, 0%)

**Usage:**
- Call-to-actions
- Alerts
- Important highlights only

---

### Color Rules
- Blue = dominant
- White = balance
- Red = minimal accent

Do NOT overuse red.

---

## 4. Typography System

### Font Style
- Clean modern sans-serif

### Hierarchy

**H1**
- Large, bold
- Used for primary headings

**H2**
- Medium, semi-bold
- Section titles

**Body Text**
- Regular weight
- High readability

**UI Text**
- Clean and slightly compact

---

## 5. Visual Design Style

### Design Principles
- Minimal
- Grid-based layout
- Balanced whitespace
- Structured alignment

### Shapes
- Rectangular layouts
- Slightly rounded corners
- No complex or abstract forms

### Icons
- Simple
- Consistent stroke width
- Minimal detailing

---

## 6. UI/UX Guidelines

### Layout
- Strong grid system
- Consistent spacing
- Clean alignment

### Components

**Buttons**
- Primary: Blue background, white text
- Secondary: White background, blue border
- Accent: Red for critical actions only

**Cards**
- Light shadow or border
- Clean padding
- Structured content

**Forms**
- Simple input fields
- Clear labels
- No clutter

---

## 7. Content & Voice

### Writing Style
- Direct
- Clear
- Professional
- No fluff or hype

### Examples

Avoid:
- "Best platform ever!!!"
- "Amazing unbelievable deals!!!"

Use:
- "Manage your properties with confidence."
- "Simple and secure property solutions."
